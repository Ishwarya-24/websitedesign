HONG KONG STADIUM


StrathAyr was contracted to construct an all new,
all weather playing surface. The irrigation system
was designed around independent head control using
Rain Bird 8005 rotors with a 25mm PEB solenoid valve
installed under each rotor. A Rain Bird SITE Control
central computerised control system operated
each sprinkler via a decoder.


Client: Hong kong stadium
